立方体相机（[name]）

创建6个远景相机并渲染到1个[page:WebGL渲染器目标(WebGLRenderTarget)Cube]对象上。

示例

[example:webgl_materials_cubemap_dynamic materials / cubemap / dynamic ]
[example:webgl_materials_cubemap_dynamic2 materials / cubemap / dynamic2 ]
[example:webgl_shading_physical shading / physical ]
//创建立方体相机 var cubeCamera = new THREE.CubeCamera( 1, 100000, 128 ); scene.add( cubeCamera ); //创建汽车 var chromeMaterial = new THREE.MeshLambertMaterial( { color: 0xffffff, envMap: cubeCamera.renderTarget } ); var car = new Mesh( carGeometry, chromeMaterial ); scene.add( car ); //更新渲染目标立方体 car.setVisible( false ); cubeCamera.position.copy( car.position ); cubeCamera.updateCubeMap( renderer, scene ); //绘制场景 car.setVisible( true ); renderer.render( scene, camera );

构造器（Constructor）

[name]( [page:number near], [page:number far], [page:number cubeResolution] )

near -- 近裁剪面距离
far -- 远裁剪面距离
cubeResolution -- 设置立方体的宽度
构造1个包含6个[page:远景相机(PerspectiveCamera) PerspectiveCameras]的立方体相机，然后渲染到1个[page:WebGL渲染器立方体目标(WebGLRenderTargetCube)]对象上。

属性（Properties）

[property:WebGLRenderTargetCube renderTarget]

生成的立方体纹理。

方法（Methods）

[method:null updateCubeMap]( [page:WebGL渲染器(WebGLRenderer) renderer], [page:Scene scene] )

renderer -- 当前的WebGL渲染器
scene -- 当前场景。
该方法用来更新渲染目标对象（renderTarget）。
