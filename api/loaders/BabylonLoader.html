[name]

加载 .babylon 资源的加载器。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager] 对象。缺省是 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager].
创建一个新的 [name]。

属性（Properties）

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] — babylon资源链接，必选参数。
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是加载的文本应答。
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url处开始加载，并传递已解析的应答内容给onLoad函数。

[method:Object3D parse]( [page:Object json] )

[page:Object json] — 要解析的JSON数据。
把一个 JSON 结构数据解析成一个 [page:3D对象(Object3D) object] 或者一个 [page:场景(Scene) scene]。
获得的对象被转换成 [page:Mesh]，带有 [page:缓存几何模型(BufferGeometry)] 和一个默认的 [page:Phong网孔材料(MeshPhongMaterial)]。
光照也被相应的解析。

例子

// 实例化一个加载器 var loader = new THREE.BabylonLoader(); // 通过URL加载一个Babylon资源 loader.load( // 资源链接 'models/babylon/skull.babylon', // 资源加载完成后的回调函数 function ( object ) { scene.add( object ); }, // 资源加载过程中的回调函数 function ( xhr ) { console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); }, // 资源下载错误的回调函数 function ( xhr ) { console.log( 'An error happened' ); } ); [example:webgl_loader_babylon]
