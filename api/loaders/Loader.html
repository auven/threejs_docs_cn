加载器（[name]）

所有加载器的基类。

构造器（Constructor）

[name]()

创建一个新的 [name]。

属性（Properties）

[property:Function onLoadStart]

将在加载开始时被调用。
缺省是一个空函数体。

[property:Function onLoadProgress]

将在加载过程中被调用。
缺省是一个空函数体。

[property:Function onLoadComplete]

将在加载结束时被调用。
缺省是一个空函数体。

[property:string crossOrigin]

实现CORS来从不同的域名（允许CORS）加载资源。

方法（Methods）

[method:Material createMaterial]( [page:object m], [page:字符串(String) texturePath] )

[page:Object m] — 创建材料的参数。
[page:字符串(String) texturePath] — 纹理的基本路径。
根据参数m创建材料。

[method:Array initMaterials]( [page:Array materials], [page:字符串(String) texturePath] )

[page:Array materials] — 创建材料的一个参数数组。
[page:字符串(String) texturePath] — 纹理的基本路径。
根据参数数组materials创建一个 [page:材料(Material)] 对象数组。 参数的索引决定材料的正确索引。

[method:String extractUrlBase]( [page:字符串(String) url] )

[page:字符串(String) url] — 从中提取基础路径的网址URL。
从网址URL中提取基础路径。
