缓存（[name]）

一个简单的缓存类，被 [page:XHR加载器(XHRLoader)] 内部所使用。

构造器（Constructor）

[name]()

创建一个新的 [name].

属性（Properties）

[property:Boolean enabled]

是否启用缓存。缺省为 *false*。

[property:Object files]

一个保存缓存数据的 [page:Object object]。

方法（Methods）

[method:null add]( [page:字符串(String) key], value )

[page:字符串(String) key] 必选参数. 一个字符串key。
[page:Object] value — 一个值。
添加一个键/值对（key/value）的缓存条目。如果key已存在，则覆盖其相应的value。

[method:null get]( [page:字符串(String) key] )

[page:字符串(String) key] 必选参数. 一个字符串key
通过键（key）获取值（value）。如果key不存在则返回未定义（*undefined*）。

[method:null remove]( [page:字符串(String) key] )

[page:字符串(String) key] 必选参数. 一个字符串key。
删除与key相关联的缓存值。

[method:null clear]()

清除缓存。
