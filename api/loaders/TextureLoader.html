纹理加载器（[name]）

用来加载一个 [page:纹理(Texture) texture] 的类。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — The [page:加载管理器(LoadingManager) loadingManager] for the loader to use. Default is [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager].
创建一个新的 [name].

属性（Properties）

[property:String crossOrigin]

缺省 — *null*.
如果不为null，把图像的 *crossOrigin* 属性赋予 *crossOrigin* 的值，在开始加载之前。

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 资源链接，必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的 [page:纹理(Texture) texture].
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数将是处理事件。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url中加载并传递已加载的 [page:纹理(Texture) texture] 给 onLoad函数。

例子

// 实例化一个加载器 var loader = new THREE.TextureLoader(); // 加载一个资源 loader.load( // 资源链接 'textures/land_ocean_ice_cloud_2048.jpg', // 资源加载完成后的回调函数 function ( texture ) { // do something with the texture var material = new THREE.MeshBasicMaterial( { map: texture } ); }, // 资源加载过程中的回调函数 function ( xhr ) { console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); }, // 资源下载出错时的回调函数 function ( xhr ) { console.log( 'An error happened' ); } ); [example:canvas_geometry_earth]
