MTL加载器（[name]）

一个用来加载.mtl格式资源的加载器，被 [page:OBJMTLLoader] 和 [page:UTF8Loader] 在内部所调用。

构造器（Constructor）

[name]( [page:字符串(String) baseUrl], [page:Object options], [page:字符串(String) crossOrigin] )

[page:字符串(String) baseUrl] — 查找后续资源的基础链接。
[page:Object options] — 用来创建材料的一些选项（side, wrap, normalizeRGB, ignoreZeroRGBs)。
[page:字符串(String) crossOrigin] — 实现CORS来从不同的域名（允许CORS）加载资源。
创建一个新的 [name].

属性（Properties）

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的 [page:MTLLoaderMaterialCreator MTLLoader.MaterialCreator] 实例。
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url处开始加载，并返回材料。

[method:MTLLoaderMaterialCreator parse]( [page:字符串(String) text] )

[page:字符串(String) text] — 要解析的mtl结构化文本。
解析一个 mtl 结构化文本并返回一个 [page:MTLLoaderMaterialCreator] 实例。
