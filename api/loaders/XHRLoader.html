XHR加载器（[name]）

使用XMLHttpRequest请求（支持异步方式）来加载资源的基础类，被大多数加载器在内部使用。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager] 对象。缺省是 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager]。
创建一个新的 [name].

属性（Properties）

[property:Cache cache]

[page:Cache cache] 实例。用来保存该加载器发出的每次请求的应答，所以每个文件被请求一次。

[property:String responseType]

用来设置响应类型。

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 资源链接，必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的应答文本。
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url开始加载，并返回包含结果数据的 [page:字符串(String) text] 应答。

[method:null setCrossOrigin]( [page:字符串(String) value] )

[page:字符串(String) value] — 实现CORS来从不同的域名（允许CORS）加载资源。

[method:null setResponseType]( [page:字符串(String) value] )

[page:字符串(String) value] — 空字符串(缺省值), "arraybuffer", "blob", "document", "json", 或者 "text"。
