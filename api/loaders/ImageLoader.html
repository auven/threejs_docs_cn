图像加载器（[name]）

用来加载一个 [page:图像(Image)].

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager] 对象。缺省是 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager].
创建一个新的 [name].

属性（Properties）

[property:String crossOrigin]

实现CORS来从不同的域名（允许CORS）加载资源。

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] — 资源链接，必选参数。
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的 [page:图像(Image) image]。
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url处开始加载资源，并返回包含数据的 [page:图像(Image) image] 对象。

[method:null setCrossOrigin]( [page:字符串(String) value] )

[page:字符串(String) value] — crossOrigin 字符串。
实现CORS来从不同的域名（允许CORS）加载资源。

例子

// 实例化一个加载器 var loader = new THREE.ImageLoader(); // 加载一个图片资源 loader.load( // 资源链接 'textures/skyboxsun25degtest.png', // 资源加载完成后的回调函数 function ( image ) { // 处理该图片 // 比如绘制在canvas上 var canvas = document.createElement( 'canvas' ); var context = canvas.getContext( '2d' ); context.drawImage( image, 100, 100 ); }, // 下载过程中的回调函数 function ( xhr ) { console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); }, // 下载出错的回调函数 function ( xhr ) { console.log( 'An error happened' ); } ); [example:webgl_shaders_ocean]
