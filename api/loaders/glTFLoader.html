glTF加载器（[name]）

一个用来加载JSON格式的.gltf资源的加载器。

glTF文件格式 是一个JSON文件格式，用来快速递交和加载3D内容。

构造器（Constructor）

[name]( )

创建一个新的 [name].

属性（Properties）

方法（Methods）

[method:Object3D load]( [page:字符串(String) url], [page:函数(Function) callback] )

[page:字符串(String) url] 必选参数
[page:函数(Function) callback] — 加载完成时被调用。参数将是一个 [page:Object]，包含已加载的 .[page:3D对象(Object3D) scene], .[page:Array cameras] 以及 .[page:Array animations]。
从url处开始加载，并使用应答内容调用callback函数。

注意

该类常和 [page:glTFAnimator THREE.glTFAnimator] 一起使用来完成动画解析。

例子

// 实例化一个加载器 var loader = new THREE.glTFLoader(); // 加载一个 glTF 资源 loader.load( // 资源链接 'models/gltf/duck/duck.json', // 资源加载完成后的回调函数 function ( object ) { scene.add( object.scene ); } ); [example:webgl_loader_gltf]
