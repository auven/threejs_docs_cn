JSON加载器（[name]）

用来加载JSON格式对象的加载器。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager]，缺省为 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager].
创建一个新的 [name].

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 必选参数.
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的应答文本。
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url处加载资源并返回 JSON 给onLoad函数。

[method:null setTexturePath]( [page:字符串(String) texturePath] )

[page:字符串(String) texturePath] — 纹理的基本路径。
设置纹理的基本路径。

[method:Object3D parse]( [page:Object json], [page:字符串(String) texturePath] )

[page:字符串(String) json] — 要解析的JSON对象。
[page:字符串(String) texturePath] — 纹理的基本路径。
解析一个 JSON 结构的数据并返回一个 [page:object] ，包含解析后的 [page:几何模型(Geometry) geometry] 和 [page:Array materials]。

例子

// 实例化一个加载器 var loader = new THREE.JSONLoader(); // 加载一个资源 loader.load( // 资源链接 'models/animated/monster/monster.js', // 资源加载完成后的回调函数 function ( geometry, materials ) { var material = new THREE.MultiMaterial( materials ); var object = new THREE.Mesh( geometry, material ); scene.add( object ); } ); [example:webgl_loader_json_blender]
