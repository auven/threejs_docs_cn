[name]

PCD 文件加载器。加载 ascii 和 二进制数据。不支持压缩过的二进制文件。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager] 对象。缺省是 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager]。
创建一个新的 [name].

属性（Properties）

[page:Boolean littleEndian]

缺省值为true，用来指定二进制数据字节序（低位优先）。

Intel X86系列CPU使用低位优先，而IBM PowerPC使用高位优先。

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 资源链接，必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的 [page:3D对象(Object3D)].
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url处开始加载，并使用应答内容调用onLoad函数。

[method:Object3D parse]( [page:Arraybuffer data],[page:字符串(String) url] )

[page:Arraybuffer data] — 要解析的二进制结构化数据。
[page:字符串(String) url] — 文件全路径名或链接。
把一个 pcd 二进制结构化数据解析为一个 [page:3D对象(Object3D)].
获取的对象被转换为 [page:Points]，使用一个 [page:缓存几何模型(BufferGeometry)] 和一个 [page:点材料(PointsMaterial)]。

例子

// 实例化一个加载器 var loader = new THREE.PCDLoader(); // 加载一个资源 loader.load( // 资源链接 'pointcloud.pcd' , // 资源加载完成后的回调函数 function ( mesh ) { scene.add( mesh ); } ); [example:webgl_loader_pcd]
