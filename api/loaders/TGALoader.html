TGA加载器（[name]）

用来加载一个 .tga [page:数据纹理(DataTexture) texture] 的类。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager] 对象。缺省是 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager]。
创建一个新的 [name].

方法（Methods）

[method:DataTexture load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 资源链接，必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的 [page:数据纹理(DataTexture)].
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数将是处理事件。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url中加载并传递已加载的 [page:数据纹理(DataTexture) texture] 给 onLoad函数。 而该 [page:数据纹理(DataTexture) texture] 也被直接返回以便立即使用（不过可能尚未完全加载）。

例子

// 实例化一个加载器 var loader = new THREE.TGALoader(); // 加载一个资源 var texture = loader.load( // 资源链接 'textures/crate_grey8.tga' // 资源加载完成后的回调函数 function ( texture ) { console.log( 'Texture is loaded' ); }, // 资源加载过程中的回调函数 function ( xhr ) { console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); }, // 资源下载出错时的回调函数 function ( xhr ) { console.log( 'An error happened' ); } ); var material = new THREE.MeshPhongMaterial( { color: 0xffffff, map: texture } ); [example:webgl_materials_texture_tga]
