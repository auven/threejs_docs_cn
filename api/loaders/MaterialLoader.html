材料加载器（[name]）

用来加载一个JSON格式的 [page:材料(Material)]。

构造器（Constructor）

[name]( [page:加载管理器(LoadingManager) manager] )

[page:加载管理器(LoadingManager) manager] — [page:加载管理器(LoadingManager) loadingManager] 对象。缺省是 [page:加载管理器(LoadingManager) THREE.DefaultLoadingManager]。
创建一个新的 [name].

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress], [page:函数(Function) onError] )

[page:字符串(String) url] 必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是已加载的 [page:材料(Material)].
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数将是处理事件。
[page:函数(Function) onError] — 如果加载有错误将调用该函数。
从url处加载资源并返回一个包含数据的[page:材料(Material)]。

[method:Material parse]( [page:Object json] )

[page:Object json] — 包含材料参数的JSON对象。
解析这个 JSON 结构化数据，并创建一个新的该类型 [page:字符串(String) json.type] 的 [page:材料(Material)]，使用定义在该json对象的参数。

例子

// 实例化一个加载器 var loader = new THREE.MaterialLoader(); // 加载一个资源 loader.load( // 资源链接 'path/to/material.json', // 资源加载完成后的回调函数 function ( material ) { object.material = material; }, // 资源加载过程中的回调函数 function ( xhr ) { console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); }, // 资源下载出错时的回调函数 function ( xhr ) { console.log( 'An error happened' ); } );
