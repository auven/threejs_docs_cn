Collada加载器（[name]）

Collada文件加载器

构造器（Constructor）

[name]()

创建一个新的 [name].

属性（Properties）

[property:Array options]

 .[page:Boolean centerGeometry] — 是否强制 [page:几何模型(Geometry)] 处于包含[page:网孔(Mesh)]对象的局部原点居中位置。
 .[page:Boolean convertUpAxis] — 是否已完成几何模型，动画和控制器的轴转换。
 .[page:Boolean subdivideFaces] — 是否强制切分为多个 [page:三角面(Face3)]。
 .[page:字符串(String) upAxis] — X, Y 或 Z
 .[page:Boolean defaultEnvMap] — 用于反射或折射材料的环境贴图。

[property:Object geometries]

已解析的 .dae 模型。

方法（Methods）

[method:null load]( [page:字符串(String) url], [page:函数(Function) onLoad], [page:函数(Function) onProgress] )

[page:字符串(String) url] 必选参数
[page:函数(Function) onLoad] — 当加载完成时被调用。参数将是包含已加载资源的[page:Object object]对象。
[page:函数(Function) onProgress] — 当加载在进行中时被调用。参数是 XmlHttpRequest 实例，包含 .[page:Integer total] 和 .[page:Integer loaded] 字节。
从url处开始加载，并使用应答内容调用onLoad函数。

[method:Object parse]( [page:Document doc], [page:函数(Function) callBack], [page:字符串(String) url] )

[page:Document doc] — 要解析的 XML 文档。
[page:函数(Function) callBack] — 解析完成时的回调函数。
[page:字符串(String) url] — 查找后续资源的基础链接。
把一个XML Document解析为一个 [page:Object object]，包含已加载的 .[page:Scene scene], .[page:Array morphs], .[page:Array skins], .[page:Array animations], .[page:Object dae]

[method:null setPreferredShading]( [page:Integer shading] )

[page:Integer shading] — 必选参数。
设置资源材料上的 .[page:Integer shading] 属性。
选项可以是 [page:材料(Material)s THREE.SmoothShading], [page:材料(Material)s THREE.FlatShading]。

[method:null applySkin]( [page:几何模型(Geometry) geometry], [page:Object instanceCtrl], [page:Integer frame] )

[page:几何模型(Geometry) geometry] — 必选参数
[page:Object instanceCtrl] — 必选参数. 这是一个collada skinController
[page:Integer frame] — 可选参数。缺省值为 40
在给定的[page:几何模型(Geometry)]上，从一个collada 皮肤控制器应用一个皮肤（vertices（顶点）, animation（动画）, bones（骨骼））。
