三维向量（[name]）

三维向量。

例子

var a = new THREE.Vector3( 1, 0, 0 ); var b = new THREE.Vector3( 0, 1, 0 ); var c = new THREE.Vector3(); c.crossVectors( a, b );

构造器（Constructor）

x -- [page:Float] 向量的x值。
y -- [page:Float] 向量的y值。
z -- [page:Float] 向量的z值。
