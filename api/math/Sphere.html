球体（[name]）

由一个中心位置和半径定义的几何球体。

构造器（Constructor）

[name]( [page:Vector3 center], [page:Float radius] )

center -- [page:Vector3]
radius -- [page:Float]

属性（Properties）

[property:Vector3 center]

[property:Float radius]

方法（Methods）

[method:Sphere set]( [page:Vector3 center], [page:Float radius] ) [page:Sphere this]

center -- [page:Vector3]
radius -- [page:Float]
设置球心和半径。

[method:Sphere applyMatrix4]( [page:Matrix4 matrix] ) [page:Sphere this]

matrix -- [page:Matrix4]
用给定的 [page:Matrix4] 对球体进行变换。

[method:Vector3 clampPoint]( [page:Vector3 point], [page:Vector3 optionalTarget] )

point -- [page:Vector3] 要夹住的点。
optionalTarget -- [page:Vector3] 返回可选的目标点。
把一个点夹进一个球体中。如果该点在球体之外，它将被夹紧到球体边缘最近的那个点。

[method:Sphere translate]( [page:Vector3 offset] ) [page:Sphere this]

offset -- [page:Vector3]
用给定的偏移量向量来平移球体的中心。

[method:Sphere clone]()

获取该球体一个新的副本。

[method:Boolean equals]( [page:Sphere sphere] )

sphere -- [page:Sphere]
检查两个球的球心和半径是否相等。

[method:Sphere setFromPoints]( [page:Array points], [page:Vector3 optionalCenter] ) [page:Sphere this]

points -- 三维点向量的数组。
optionalCenter -- 可选的球体中心点位置。
计算一组点的最小包围球。如果给定了*optionalcenter*，则用来作为球体的中心。否则，包围盒的中心通过计算而来。

[method:Float distanceToPoint]( [page:Vector3 point] )

point -- [page:Vector3]
返回从球体的边界到点的最接近的距离。如果球体包含了点，距离将是负的。

[method:Box getBoundingBox]( [page:Box optionalTarget] )

optionalTarget -- [page:Box]
获取一个球体的包围盒，可选目标（optionalTarget）参数用来设置结果。

[method:Boolean containsPoint]( [page:Vector3 point] )

point -- [page:Vector3]
检查球体是否包含给定的点（包括球体的边缘）。

[method:Sphere copy]( [page:Sphere sphere] )

sphere -- 要拷贝的[page:Sphere]对象。
将传递的球体参数的值复制到当前球体。

[method:Boolean intersectsSphere]( [page:Sphere sphere] )

sphere -- [page:Sphere]
检查两个球体是否相交。

[method:Boolean empty]()

检查球体是否为空（半径设置为0）。
