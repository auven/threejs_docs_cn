样条（[name]）

代表一个样条（spline）。

构造器（Constructor）

[name]( [page:Array points] )

使用*points*初始化该spline，spline将穿过这些点。

属性（Properties）

[property:Array points]

方法（Methods）

[method:null initFromArray]( [page:Array a] )

a — 包含x,y,z坐标的点的数组。
使用一组点来初始化样条数据。a[n]表示第n个点的取值v，其中v[0],v[1],v[2]分别是该点的x,y,z坐标。

[method:Vector3 getPoint]( [page:Integer k] )

k — 点的索引。
返回*k*处的插值点。

[method:Array getControlPointsArray]( )

返回控制点数组。

[method:Object getLength]( [page:Integer nSubDivisions] )

nSubDivisions — 控制点之间的分割数。默认为 *100*。
返回一个包含2个属性的对象。属性 .[page:Number total] 包含使用nSubDivisions时的样条长度。 属性 .[page:Array chunkLength] 包含该样条每一段的长度数组。

[method:null reparametrizeByArcLength]( [page:Float samplingCoef] )

samplingCoef — 样条点之间使用多少中间值。
修改样条曲线，使它看起来类似于原来的，但点沿着曲线的分布更均匀。
这是通过对原样条进行重采样来完成的，采样密度由参数*samplingcoef*所控制。要注意的是，更密集的采样不一定是好的：如果采样太高，你可能在曲率得到奇怪的扭结。
