四维矩阵（[name]）

一个 4x4 矩阵。

例子

// 绕3个轴旋转的简单操作。 var m = new THREE.Matrix4(); var m1 = new THREE.Matrix4(); var m2 = new THREE.Matrix4(); var m3 = new THREE.Matrix4(); var alpha = 0; var beta = Math.PI; var gamma = Math.PI/2; m1.makeRotationX( alpha ); m2.makeRotationY( beta ); m3.makeRotationZ( gamma ); m.multiplyMatrices( m1, m2 ); m.multiply( m3 );

构造器（Constructor）

[name]()

创建和初始化一个3x3的单位矩阵。

属性（Properties）

[property:Float32Array elements]

矩阵值的列主列表。
