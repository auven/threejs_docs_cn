[name]

Euler Angles.

Euler angles describe a rotation transformation by rotating an object on its various axes in specified amounts per axis, and a specified axis order. (More information on Wikipedia)

例子

var a = new THREE.Euler( 0, 1, 1.57, 'XYZ' ); var b = new THREE.Vector3( 1, 0, 1 ); b.applyEuler(a);

构造器（Constructor）

[name]( [page:Float x], [page:Float y], [page:Float z], [page:字符串(String) order] )

x -- [page:Float] x轴角度（单位为弧度）
y -- [page:Float] y轴角度（单位为弧度）
z -- [page:Float] z轴角度（单位为弧度）
order -- [page:字符串(String)] 一个用来表示旋转顺序的字符串，默认为 'XYZ'（必须是大写）。
一个用来变换的欧拉角。

[property:Float y]

方法（Methods）

[method:Euler set]( [page:Float x], [page:Float y], [page:Float z], [page:字符串(String) order] ) [page:欧拉角(Euler) this]

x -- [page:Float] x轴角度（单位为弧度）
y -- [page:Float] y轴角度（单位为弧度）
z -- [page:Float] z轴角度（单位为弧度）
order -- [page:字符串(String)] 坐标轴的顺序，默认为 'XYZ'（必须是大写）。
设置该欧拉角变换的角度。

[method:Euler copy]( [page:欧拉角(Euler) euler] ) [page:欧拉角(Euler) this]

拷贝 *euler* 的值给该欧拉角。

[method:Euler setFromRotationMatrix]( [page:Matrix4 m], [page:字符串(String) order] ) [page:欧拉角(Euler) this]

m -- [page:Matrix4] 假设矩阵上面的3x3部分是一个纯旋转变换（也就是不发生缩放）
order -- [page:字符串(String)] 坐标轴的顺序，默认为 'XYZ'（必须是大写）。
以基于order所指定方向的一个纯旋转矩阵来设置欧拉角变换的角度。

[method:Euler setFromQuaternion]( [page:四元数(Quaternion) q], [page:字符串(String) order] ) [page:欧拉角(Euler) this]

q -- [page:四元数(Quaternion)] quaternion must be normalized
order -- [page:字符串(String)] 坐标轴的顺序，默认为 'XYZ'（必须是大写）。
以基于order所指定方向的一个四元数来设置欧拉角变换的角度。

[method:Euler reorder]( [page:字符串(String) newOrder] ) [page:欧拉角(Euler) this]

使用newOrder新建一个四元数来重置欧拉角。
警告: 这将弃用旋转信息。

[method:Euler setFromVector3]( [page:Vector3 vector], [page:字符串(String) order] ) [page:欧拉角(Euler) this]

vector -- [page:Vector3]. order -- [page:字符串(String)] Order of axes, defaults to 'XYZ' (must be upper case)
使用可选的Vector3参数来设置欧拉角的XYZ参数，并用order参数设置欧拉角的order属性。

[method:Vector3 toVector3]()

使用一个Vector3返回欧拉角的XYZ属性。

[method:Euler fromArray]( [page:Array array] ) [page:欧拉角(Euler) this]

array -- 长度为3或4的一个[page:Array]。array[3] 是一个可选的order参数。
把该欧拉角的x角度设置为 array[0].
把该欧拉角的y角度设置为 array[1].
把该欧拉角的z角度设置为 array[2].
设置该欧拉角的order为array[3]，可选。

[method:Array toArray]( [page:Array array] )

array -- 把欧拉角转换成一个数组。
返回一个数组：[x, y, z, order]

[method:Boolean equals]( [page:欧拉角(Euler) euler] )

判断该euler和参数*euler*是否相等。

[method:Euler clone]()

克隆一个欧拉角。
