[name]

从一个原点向特定方向发射的射线。

构造器（Constructor）

[name]( [page:Vector3 origin], [page:Vector3 direction] )

origin -- [page:Vector3] [page:射线(Ray)]原点。
direction -- [page:Vector3] [page:射线(Ray)]方向。这个参数必须是归一化的（使用 [page:Vector3].normalize方法）。
使用给定值来初始化origin和direction属性。

属性（Properties）

[property:Vector3 origin]

[page:射线(Ray)]原点。

[property:Vector3 direction]

[page:射线(Ray)]方向。这个参数必须是归一化的（使用 [page:Vector3].normalize方法）。

方法（Methods）

[method:Ray applyMatrix4]( [page:Matrix4 matrix4] )

matrix4 -- [page:Matrix4] 用来变换该射线的 [page:Matrix4] 。
使用[page:Matrix4]来变换[page:Ray]。

[method:Vector3 at]( [page:Float t], [page:Vector3 optionalTarget] = null ) [page:Vector3]

t -- [page:Float] 该位置到射线的距离。
optionalTarget -- [page:Vector3] 如果设置了该参数，则获取射线方向上的一个位置；否则新建一个 [page:Vector3] 对象。
获取沿射线方向指定距离的一个 [page:Vector3] 位置点。

[method:Ray clone]()

克隆该 [page:Ray].

[method:Vector3 closestPointToPoint]( [page:Vector3 point], [page:Vector3 optionalTarget] )

point -- [page:Vector3] 用来获取射线上最近点的参照点位置。
optionalTarget -- [page:Vector3] 如果设置了该参数，用来获取返回值；否则新建一个 [page:Vector3] 对象。
获取沿着射线方向和给定 [page:Vector3] 距离最近的点。

[method:Ray copy]( [page:Ray ray] )

ray -- 拷贝源。
拷贝给定ray对象的属性并返回。

[method:Float distanceSqToSegment]( [page:Vector3 v0], [page:Vector3 v1], [page:Vector3 optionalPointOnRay] = null, [page:Vector3 optionalPointOnSegment] = null ) [page:Float]

v0 -- [page:Vector3] 线段起点。 v1 -- [page:Vector3] 线段终点。 optionalPointOnRay -- [page:Vector3] 如果给定了该参数，则用来接收距离线段最近的射线上的点。 optionalPointOnSegment -- [page:Vector3] 如果给定了该参数，则用来接收距离射线最近的线段上的点。
获取一个线段和该射线之间的平方距离。

[method:Float distanceToPlane]( [page:Plane plane] )

plane -- [page:Plane] 计算距离的目标平面。
获取原点到给定平面的距离，或者为*null*，如果射线并不和平面相交。

[method:Float distanceToPoint]( [page:Vector3 point] )

point -- [page:Vector3] 计算距离的目标点。
获取目标点和射线之间的最短距离。

[method:Float distanceSqToPoint]( [page:Vector3 point] )

point -- [page:Vector3] 计算距离的目标点。
获取目标点和射线之间的最短平方距离。

[method:Boolean equals]( [page:Ray ray] )

ray -- [page:Ray] 要比较的射线。
返回两条射线是否具有相同的偏移和方向。

[method:Vector3 intersectBox]( [page:Box3 box], [page:Vector3 optionalTarget] = null ) [page:Vector3]?

box -- [page:Box3] 相交的box3盒子。
optionalTarget -- [page:Vector3] 用来保存结果的[page:Vector3], 默认为新建一个 [page:Vector3] 对象。
使得射线和一个盒子相交，返回交点。如果没有交点，则返回 *null*。

[method:Vector3 intersectPlane]( [page:Plane plane], [page:Vector3 optionalTarget] = null ) [page:Vector3]?

plane -- [page:Plane] 相交的平面。
optionalTarget -- [page:Vector3] 用来保存结果的[page:Vector3], 默认为新建一个 [page:Vector3] 对象。
使得射线和一个平面相交，返回交点。如果没有交点，则返回 *null*。
function ( a, b, c, backfaceCulling, optionalTarget )

[method:Vector3 intersectTriangle]( [page:Vector3 a], [page:Vector3 b], [page:Vector3 c], [page:Boolean backfaceCulling], [page:Vector3 optionalTarget] = null ) [page:Vector3]?

a, b, c -- [page:Vector3] 用来定义三角形的3个点。
backfaceCulling -- [page:Boolean] Whether to use backface culling.
optionalTarget -- [page:Vector3] 用来保存结果的[page:Vector3], 默认为新建一个 [page:Vector3] 对象。
使得射线和一个三角形相交，返回交点。如果没有交点，则返回 *null*。

[method:Boolean intersectsBox]( [page:Box3 box] )

box -- [page:Box3] 相交的box3盒子。
返回是否射线和盒子相交，这是一个布尔值。

[method:Boolean intersectsPlane]( [page:Plane plane] )

plane -- [page:Plane] 相交的平面。
返回是否射线和平面相交。

[method:Boolean intersectsSphere]( [page:Sphere sphere] )

sphere -- [page:Sphere] 相交的球体。
返回是否射线和球体相交。

[method:Ray recast]( [page:Float t] )

t -- 需要插值的沿着射线方向的距离。
按给定的距离沿着射线方向迁移原点。

[method:Ray set]( [page:Vector3 origin], [page:Vector3 direction] )

origin -- [page:Vector3] [page:射线(Ray)]原点。
direction -- [page:Vector3] [page:射线(Ray)]方向。这个参数必须是归一化的（使用 [page:Vector3].normalize方法）。
拷贝参数给origin和direction属性。

[method:Ray lookAt]( [page:Vector3 v] )

v -- [page:Vector3] 观察点位置。
把射线方向调整为世界坐标中该vector所在的点。
