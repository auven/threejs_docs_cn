点材料（[name]）

[page:Points particle] 粒子系统所使用的缺省材料。

构造器（Constructor）

[name]( [page:Object parameters] )

参数是一个使用一个或多个属性定义材料外观的对象。
color — 16进制的粒子颜色。默认为 0xffffff.
map — 一个 [page:纹理(Texture) texture]。如果被定义，那么一个点（point）从纹理中获取颜色数据。缺省为null。
size — 定义粒子的大小。默认为1.0。
sizeAttenuation — 启用/禁用随距离而发生尺寸衰减。
vertexColors — 定义材料是否使用顶点颜色。默认是false。
fog — 定义材质颜色是否受全局雾设置的影响。默认是true。

属性（Properties）

通用属性请查看基类 [page:材料(Material)] 。

[property:Number color]

设置粒子的颜色。默认值是 0xffffff.

[property:Texture map]

使用纹理数据设置粒子的颜色。

[property:Number size]

设置粒子的大小。默认为1.0。

[property:Boolean sizeAttenuation]

指明粒子大小是否随距离而变。默认为true。

[property:Integer vertexColors]

定义顶点如何着色。可用值为 THREE.NoColors, THREE.FaceColors 和 THREE.VertexColors。缺省值为 THREE.NoColors.
当使用一些特定渲染器时，此设置可能没有任何效果。

[property:Boolean fog]

定义材质颜色是否受全局雾设置的影响。
当使用一些特定渲染器时，此设置可能没有任何效果。比如，该属性被 [page:Canvas渲染器(CanvasRenderer) Canvas] 所忽略，但是对 [page:WebGL渲染器(WebGLRenderer) WebGL] 有效。
