精灵材料（[name]）

使用 [page:Sprite] 的一种材料。

构造器（Constructor）

[name]( [page:Object parameters] )

parameters -- 定义默认属性的对象
color - sprite的颜色。
map - 纹理贴图
rotation - sprite的旋转
fog - 是否使用场景雾。

属性（Properties）

通用属性请查看基类 [page:材料(Material)] 。

[property:Color color]

纹理通过这个颜色增强。默认的是0xffffff。

[property:Texture map]

纹理贴图。缺省为空。

[property:Radians rotation]

以弧度为单位的sprite的转动。默认为0。

[property:boolean fog]

定义材质颜色是否受全局雾设置的影响。默认是true。
