多种材料（[name]）

一种用于定义相同几何模型的多个材料的材料。 模型使用 [page:Face3 faces materialindex] 来决定哪个面使用哪个材料。 其中materialindex对应于材料数组中的材料索引值。

构造器（Constructor）

[name]( [page:Array materials] )

materials -- 模型的材料数组。
创建一个多种材料。

属性（Properties）

[property:Array materials]

获取或设置模型的材料。
