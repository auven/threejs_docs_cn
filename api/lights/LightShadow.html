属性（Properties）

[property:Camera camera]

阴影的空间观察视角。

[property:Float bias]

在本文中出现的词汇：阴影映射、阴影贴图和Shadow Map同义。

阴影映射偏差，当决定一个曲面是否处于阴影中时，从正规化深度中增加或减少多少。
缺省值 — *0*.

[property:Float radius]

阴影半径
缺省值 - *0*.

[property:Vector2 mapSize]

存储在一个[page:Vector2 Vector2]中的阴影贴图的宽度和高度。
Default — *( 512, 512 )*.

[property:WebGLRenderTarget map]

使用内置相机生成的深度图；超出像素深度的位置是在阴影中。在渲染期间内部计算。

[property:Matrix4 matrix]

在相机空间产生投影的模型，用来计算阴影贴图中的位置和深度。保存于一个[page:四维矩阵(Matrix4) Matrix4]对象中。在渲染期间内部计算。
