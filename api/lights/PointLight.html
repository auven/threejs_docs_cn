构造器（Constructor）

[name]( [page:Integer color], [page:Float intensity], [page:Number distance], [page:Float decay] )

[page:Integer color] — 颜色的RBG数值。
[page:Float intensity] — 光强的数值。
[page:Number distance] -- 光强为0处到光源的距离，0表示无穷大。
[page:Float decay] -- 沿着光照距离的衰退量。
在场景中的特定位置创建一个光源。光线照射在各个方向上（好比一个灯泡）。

属性（Properties）

通用属性可查看基类 [page:光照(Light) Light] 。

[property:Float power]

光功率。
在“物理正确”模式中，表示以"流明（光通量单位）"为单位的光功率。
Default - *4PI*。

[property:Float distance]

如果非0，光照将沿着光源处到该距离处线性衰减为零。
缺省值 — *0.0*，表示光照一直到无穷远才衰减为零。

[property:Float decay]

沿着光照距离的衰退量。
在“物理正确”模式中，decay = 2将实现现实世界的光衰减。
缺省值 — *1*。

[property:LightShadow shadow]

此属性存储用来渲染光照阴影的所有相关信息。
