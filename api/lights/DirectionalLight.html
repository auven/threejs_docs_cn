构造器（Constructor）

[name]( [page:Integer hex], [page:Float intensity] )

[page:Integer hex] -- 光源颜色的RGB数值。
[page:Float intensity] -- 光源强度的数值。
创建一个光照，从一个特定的方向，而不是从一个特定的位置。这个光看起来就像光源位于无限远处，因此它产生的光线都是平行的。 最好的类比是一个像太阳一样的光源：太阳是如此遥远，所有的阳光照射到物体上都几乎来自同一个角度。

属性（Properties）

通用属性可查看基类 [page:光照(Light) Light] 。

[property:Object3D target]

用于阴影相机定位的目标。

[property:DirectionalLightShadow shadow]

该属性存储渲染光照阴影的所有相关信息。

[property:Boolean castShadow]

如果设置为*true*，光照将产生动态阴影。*警告*：这是代价高昂的，需要调整以使得阴影看起来是正确的。
缺省值 — *false*.
