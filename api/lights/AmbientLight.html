环境光（[name]）

这种光的颜色被应用到全局范围内的所有对象。

构造器（Constructor）

[name]( [page:Integer color], [page:Float intensity] )

[page:Integer color] — 光源颜色的RGB数值。
[page:Float intensity] -- 光源强度的数值。
这将创建一个具有给定颜色和强度的环境光。
