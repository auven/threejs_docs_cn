三角面（[name]）

三角面（Triangle face）。

代码示例

var normal = new THREE.Vector3( 0, 1, 0 ); var color = new THREE.Color( 0xffaa00 ); var face = new THREE.Face3( 0, 1, 2, normal, color, 0 );

构造器（Constructor）

[name]( [page:Integer a], [page:Integer b], [page:Integer c], [page:Vector3 normal], [page:Color color], [page:Integer materialIndex] )

a — 顶点索引A。
b — 顶点索引B。
c — 顶点索引C。
normal — 面法向量或顶点法向量数组。
color — 面颜色或顶点颜色的数组。
materialIndex — 材料索引。

属性（Properties）

[property:Integer a]

顶点索引A。

[property:Integer b]

顶点索引B。

[property:Integer c]

顶点索引C。

[property:Vector3 normal]

面法向量。

[property:Color color]

面颜色。

[property:Array vertexNormals]

3个顶点法向量的数组。

[property:Array vertexColors]

3个顶点颜色的数组。

[property:Integer materialIndex]

材料索引 (指向 [page:多种材料(MultiMaterial) MultiMaterial.materials]).

方法（Methods）

[method:Face3 clone]()

创建一个新的Face3克隆对象。
