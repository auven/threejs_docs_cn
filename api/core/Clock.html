时钟（[name]）

跟踪时间的对象。

构造器（Constructor）

[name]( [page:Boolean autoStart] )

autoStart — 用来初始化autoStart属性，表示是否自动启动时钟。默认是true。

属性（Properties）

[property:Boolean autoStart]

如果设置为true，当第一次更新被调用时，自动启动时钟。

[property:Float startTime]

当时钟运行时，该属性用来保存时钟的起始时间。
这个是从1 January 1970 00:00:00 UTC开始的毫秒数。

[property:Float oldTime]

当时钟运行时，该属性用来保存最近一次更新的时间。
这个是从1 January 1970 00:00:00 UTC开始的毫秒数。

[property:Float elapsedTime]

当时钟运行时，该属性用来保存起始时间和最近一次更新时间之间的时间间隔。
这个是从1 January 1970 00:00:00 UTC开始的毫秒数。

[property:Boolean running]

这个属性跟踪时钟是否正在运行。

方法（Methods）

[method:null start]()

启动时钟。

[method:null stop]()

停止时钟。

[method:Float getElapsedTime]()

获取时钟已运行时间。

[method:Float getDelta]()

获取时钟增量，也就是该函数本次调用和上次调用之间的时间间隔。
