3D对象（[name]）

3D场景中图形对象的基类。

构造器（Constructor）

[name]()

该构造函数没有参数。

属性（Properties）

[property:Integer id]

只读 – 此对象实例的唯一编号。

[property:String uuid]

该对象的[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] 由程序自动分配，所以不可编辑。

[property:String name]

对象的可选名称（不需要是唯一的）。

[property:Object3D parent]

该对象在场景图中的父对象。

[property:Object3D children]

该对象的子对象数组。

[property:Vector3 position]

对象的局部位置。

[property:Euler rotation]

对象的局部旋转，单位为弧度。 (欧拉角（Euler angles）)。

[property:Vector3 scale]

对象的局部缩放因子。

[property:Vector3 up]

空间的向上方向。缺省是 THREE.Vector3( 0, 1, 0 )。

[property:Matrix4 matrix]

局部变换。

[property:Quaternion quaternion]

使用[page:四元数(Quaternion) Quaternion]表示的对象局部旋转。

[property:Boolean visible]

是否可见，如果为true，则对象被渲染。
缺省值 – true

[property:Boolean castShadow]

是否被渲染到阴影映射中。
缺省值 – false

[property:Boolean receiveShadow]

材料接受阴影效果。
缺省值 – false

[property:Boolean frustumCulled]

当设置为true时，每一帧都会检查该对象是否在相机的视椎体中。否则，即使它是不可见的，对象也会得到绘制。
缺省值 – true

[property:Boolean matrixAutoUpdate]

当设置为true时会自动更新矩阵，包括计算位置矩阵（旋转或四元数），逐帧缩放，也会重新计算世界矩阵（matrixWorld）属性。
缺省值 – true

[property:Boolean matrixWorldNeedsUpdate]

当设置为true时，会计算该帧中的世界矩阵属性，然后重置该属性为false。
缺省值 – false

[property:object userData]

可以用来存储该Object3D对象的自定义数据，它不应该保存有函数的引用，因为这些不会被克隆。

[property:Matrix4 matrixWorld]

对象的全局变换。如果该Object3D没有父对象，那么它和局部变换相同。

方法（Methods）

该类可以使用方法 [page:事件分发器(EventDispatcher) EventDispatcher]

[method:null applyMatrix]( [page:Matrix4 matrix] )

matrix - 矩阵
使用矩阵变换更新对象的位置、旋转和大小。

[method:null translateX]( [page:Float distance] )

distance - 距离。
沿X轴平移对象。

[method:null translateY]( [page:Float distance] )

distance - 距离。
沿Y轴平移对象。

[method:null translateZ]( [page:Float distance] )

distance - 距离。
沿Z轴平移对象。

[method:null rotateX]( [page:Float rad] )

rad - 以弧度为单位的旋转角度
在局部空间中绕X轴旋转对象。

[method:null rotateY]( [page:Float rad] )

rad - 以弧度为单位的旋转角度
在局部空间中绕Y轴旋转对象。

[method:null rotateZ]( [page:Float rad] )

rad - 以弧度为单位的旋转角度
在局部空间中绕Z轴旋转对象。

[method:Vector3 localToWorld]( [page:Vector3 vector] )

vector - 一个局部向量
从局部空间到世界空间（全局空间）的向量转换。

[method:Vector3 worldToLocal]( [page:Vector3 vector] )

vector - 一个全局向量
从世界空间到局部空间的向量转换。

[method:null lookAt]( [page:Vector3 vector] )

vector - 一个世界向量观察点
旋转模型以面对观察点。

[method:null add]( [page:3D对象(Object3D) object], ... )

object - 一个对象
添加 *object* 为该对象的子对象。可以添加任意数量的对象。

[method:null remove]( [page:3D对象(Object3D) object], ... )

object - 一个对象
删除该对象中的 *object* 子对象。可以删除任意数量的对象。

[method:null traverse]( [page:函数(Function) callback] )

callback - 第一个参数为Object3D对象的回调函数。
在这个对象和所有的子对象上执行回调。

[method:null traverseVisible]( [page:函数(Function) callback] )

callback - 第一个参数为Object3D对象的回调函数。
和 traverse 方法类似，但是这个callback将只在可见对象上执行。 不可见对象的后代不会被遍历。

[method:null traverseAncestors]( [page:函数(Function) callback] )

callback - 第一个参数为Object3D对象的回调函数。
执行所有祖先的回调。

[method:null updateMatrix]()

更新对象的局部变换。

[method:null updateMatrixWorld]( [page:Boolean force] )

更新对象和它的子对象的全局变换。

[method:Object3D clone]( [page:Boolean recursive] )

recursive -- 如果为true，对象的后代也会被克隆（概念上类似于C语言的深度拷贝）。默认是true。
返回该对象的克隆以及可选的所有后代对象。

[method:Object3D getObjectByName]( [page:字符串(String) name] )

name -- 和子对象的Object3d.name属性匹配的字符串
搜索对象的子对象，并返回第一个和name匹配的对象。

[method:Object3D getObjectById]( [page:Integer id] )

id -- 对象实例的唯一编号
搜索对象的子对象，并返回第一个和id匹配的对象。

[method:Vector3 getWorldPosition]( [page:Vector3 optionalTarget] )

optionalTarget — 用来保存结果的目标对象。否则，实例化一个新的`Vector3`对象。（可选）
返回一个表示对象在世界空间中的位置的向量。

[method:Quaternion getWorldQuaternion]( [page:四元数(Quaternion) optionalTarget] )

optionalTarget — 用来保存结果的目标对象。否则，实例化一个新的`Quaternion`对象。
返回一个表示在世界空间中对象的旋转的四元数。

[method:Euler getWorldRotation]( [page:欧拉角(Euler) optionalTarget] )

optionalTarget — 用来保存结果的目标对象。否则，实例化一个新的`Euler`对象。
返回一个表示在世界空间中对象的旋转的欧拉角。

[method:Vector3 getWorldScale]( [page:Vector3 optionalTarget] )

optionalTarget — 用来保存结果的目标对象。否则，实例化一个新的`Vector3`对象。（可选）
返回用在世界空间中每个轴的对象上的缩放因子的向量。

[method:Vector3 getWorldDirection]( [page:Vector3 optionalTarget] )

optionalTarget — 用来保存结果的目标对象。否则，实例化一个新的`Vector3`对象。（可选）
返回一个向量，表示在世界空间物体的正Z轴方向。

[method:Object3D translateOnAxis]( [page:Vector3 axis], [page:Float distance] )

axis -- 对象空间中的正规化（也称之为归一化）轴向量。
distance -- 平移距离。
在对象空间中沿轴线平移一个物体。这里的轴假定为已正规化。

[method:Object3D rotateOnAxis]( [page:Vector3 axis], [page:Float angle] )

axis -- 对象空间中的正规化轴向量。
angle -- 以弧度为单位的角度。
在对象空间中绕轴线旋转一个物体。这里的轴假定为已正规化。

[method:Array raycast]( [page:Raycaster raycaster], [page:数组(Array) intersects] )

获取光线投射和该对象之间交点的抽象方法。子类（Subclasses）如 [page:网孔(Mesh)], [page:线条(Line)], 和 [page:点(Points)] 实现了该方法以参与光线投射（raycasting）。
