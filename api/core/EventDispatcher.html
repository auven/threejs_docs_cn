事件调度器（[name]）

自定义对象的JavaScript事件。

构造器（Constructor）

[name]()

创建一个 EventDispatcher 对象。

方法（Methods）

[method:null addEventListener]( [page:字符串(String) type], [page:函数(Function) listener] )

type - 侦听的事件类型。
listener - 侦听器，即事件被触发时调用的函数。
为某类型事件添加侦听器。

[method:Boolean hasEventListener]( [page:字符串(String) type], [page:函数(Function) listener] )

type - 侦听的事件类型。
listener - 侦听器，即事件被触发时调用的函数。
检测某类型事件是否已添加侦听器。

[method:null removeEventListener]( [page:字符串(String) type], [page:函数(Function) listener] )

type - 要删除的侦听器类型。
listener - 要删除的侦听器函数。
删除某类型事件的侦听器。

[method:null dispatchEvent]( [page:object event] )

event - 被触发的事件。
触发某类型事件。
