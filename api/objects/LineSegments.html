线段（[name]）

成对的点之间的连线。

构造器（Constructor）

[name]( [page:几何模型(Geometry) geometry], [page:材料(Material) material] )

geometry — 代表每一个线段的顶点对。
material — 线条材质，默认为 [page:基础线条材料(LineBasicMaterial) LineBasicMaterial].
如果没有提供材料参数，将使用随机材料。

属性（Properties）

[property:Geometry geometry]

代表每一个线段的顶点对。

[property:Material material]

线条材质。

方法（Methods）

[method:Array raycast]( [page:Raycaster raycaster], [page:Array intersects] )

得到该线段和光线投射之间的交点。[page:Raycaster.intersectObject] 将调用该方法。

[method:LineSegments clone]()

返回该线段对象及其后代的克隆。
