镜头光晕（[name]）

创建一个模拟的跟踪光线的镜头光晕。

注意：[page:WebGL渲染器(WebGLRenderer)] 必须把 *alpha* 设置为 *true*，镜头光晕才工作。

例子

[example:webgl_lensflares lensflares]

构造器（Constructor）

[name]( [page:纹理(Texture) texture], [page:Float size], [page:Float distance], [page:材料(Material)s blending], [page:Color color] )

texture -- THREE.Texture （可选）
size -- 像素尺寸 (-1 = 使用 texture.width)
distance -- 介于 (0-1) 的光源距离（0 = 在光源处）
blending -- [page:材料(Material)s 混合模式] - 缺省为 THREE.NormalBlending
color -- 镜头光晕的颜色。
构造一个镜头光晕对象。如果还设置了一个纹理，则自动添加一个镜头光晕到lensFlares数组中。

属性（Properties）

[property:array lensFlares]

通过 [page:LensFlare.add] 方法添加的镜头光晕数组。

[property:Vector3 positionScreen]

镜头光晕在屏幕上的位置。

[property:Function customUpdateCallback]

一个自定义的更新回调。

方法（Methods）

[method:null add]( [page:纹理(Texture) texture], [page:Float size], [page:Float distance], [page:材料(Material)s blending], [page:Color color] )

添加一个镜头光晕。参考构造器的参数描述。

[method:null updateLensFlares]()

基于 [page:LensFlare.positionScreen positionScreen] 属性更新镜头光晕。

[method:Lens Flare clone]()

返回该镜头光晕对象及其后代的克隆。
