细致度（[name]）

细节的级别 - 根据距离显示或多或少的网格模型。

例子

[example:webgl_lod LOD]
var lod = new THREE.LOD(); //Create 5 levels of spheres for( var i = 0; i < 5; i++ ) { var geometry = new THREE.IcosahedronGeometry( 10, 5 - i ) new THREE.Mesh( geometry, material ); lod.addLevel( mesh, i * 50 ); }

构造器（Constructor）

[name]()

属性（Properties）

[property:array levels]

一个 [page:object level] 对象数组。
level对象拥有两个属性。
distance -- 显示此级别详细信息的距离。
object -- 要显示的3d对象。

方法（Methods）

[method:null addLevel]( [page:3D对象(Object3D) mesh], [page:Float distance] )

mesh -- 要显示的3d对象。
distance -- 显示此级别详细信息的距离。
添加一个网孔，将显示在大于等于特定距离处。通常越远，网孔细致度越低。

[method:Object3D getObjectForDistance]( [page:Float distance] )

获取一个大于给定距离的第一个网孔对象。

[method:null update]( [page:Camera camera] )

camera -- 当前相机。
更新基于相机距离的细致度的可见性。

[method:LOD clone]()

返回该LOD对象及其距离相关对象的克隆。
