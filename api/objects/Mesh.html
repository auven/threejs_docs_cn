网孔（[name]）

网孔对象的基类，如 [page:MorphAnimMesh] 和 [page:蒙皮网孔(SkinnedMesh)]。

本文档中，为了避免和“grid”网格冲突，使用“网孔”一词来翻译“Mesh”。

例子

var geometry = new THREE.BoxGeometry( 1, 1, 1 ); var material = new THREE.MeshBasicMaterial( { color: 0xffff00 } ); var mesh = new THREE.Mesh( geometry, material ); scene.add( mesh );

构造器（Constructor）

[name]( [page:几何模型(Geometry) geometry], [page:材料(Material) material] )

geometry — An instance of [page:几何模型(Geometry)].
material — An instance of [page:材料(Material)] (optional).

属性（Properties）

[property:Geometry geometry]

一个 [page:几何模型(Geometry)] 实例，用来定义对象的结构。

[property:Material material]

一个 [page:材料(Material)] 实例，用来定义对象的外观。缺省是一个启用线框模式和随机颜色的 [page:基础网孔材料(MeshBasicMaterial)] 。

[property:Array morphTargetInfluences]

一个权重值数组，取值从0到1，指定变形程度。 默认为undefined，在调用 [page:Mesh.updateMorphTargets updateMorphTargets] 方法时，会被重置为一个空数组。

[property:Array morphTargetDictionary]

一种基于morphTarget.name属性的morphTargets对象词典。 默认为undefined，在调用 [page:Mesh.updateMorphTargets updateMorphTargets] 方法时，会被重构。

[property:Integer morphTargetBase]

指定该索引处的morphTarget为基础变形。 默认为undefined，在调用 [page:Mesh.updateMorphTargets updateMorphTargets] 方法时，会被重置为 -1。

方法（Methods）

[method:Integer getMorphTargetIndexByName]( [page:字符串(String) name] )

name — 一个变形标的（morph target）对象的名称。
返回一个给定名字的变形标的的索引。

[method:null updateMorphTargets]()

更新 morphtargets 使其不对对象产生影响。重置 [page:Mesh.morphTargetForcedOrder morphTargetForcedOrder], [page:Mesh.morphTargetInfluences morphTargetInfluences], [page:Mesh.morphTargetDictionary morphTargetDictionary], 和 [page:Mesh.morphTargetBase morphTargetBase] 属性。

[method:Array raycast]( [page:Raycaster raycaster], [page:Array intersects] )

得到光线投射和网孔之间的交叉点。[page:Raycaster.intersectObject] 将调用该方法。

[method:Mesh clone]()

返回这个网孔对象及其子对象的克隆。
