骨骼（[name]）

骨骼（bone） 是 [page:骨架(Skeleton)]的一部分。骨架（Skeleton）转而被用于 [page:蒙皮网孔(SkinnedMesh)]。 骨骼几乎等同于一个空的 [page:3D对象(Object3D)]。

例子

var root = new THREE.Bone(); var child = new THREE.Bone(); root.add( child ); child.position.y = 5;

构造器（Constructor）

[name]( [page:蒙皮网孔(SkinnedMesh) skin] )

skin — (可选) 该骨骼所属的 [page:蒙皮网孔(SkinnedMesh)]。

属性（Properties）

[property:SkinnedMesh skin]

一个可选的 [page:蒙皮网孔(SkinnedMesh)] 对象引用。

方法（Methods）

[method:Bone clone]()

返回此骨骼对象及其后代的克隆。
