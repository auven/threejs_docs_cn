骨架（[name]）

使用一个 [page:骨骼(Bone) bones] 数组来创建一个骨架，可被一个 [page:蒙皮网孔(SkinnedMesh)] 所使用。只对WebGL渲染器有用。

例子

// 创建一个胳膊 var bones = []; var shoulder = new THREE.Bone(); var elbow = new THREE.Bone(); var hand = new THREE.Bone(); shoulder.add( elbow ); elbow.add( hand ); bones.push( shoulder ); bones.push( elbow ); bones.push( hand ); shoulder.position.y = -5; elbow.position.y = 0; hand.position.y = 5; var armSkeleton = new THREE.Skeleton( bones ); // 查看 THREE.SkinnedMesh 中结合网孔使用的一个示例。

构造器（Constructor）

[name]( [page:Array bones], [page:Array boneInverses], [page:Boolean useVertexTexture] )

bones — [page:骨骼(Bone) bones] 数组。
boneInverses — (可选) 一个 [page:Matrix4 Matrix4s] 数组。
useVertexTexture — (可选) 是否在着色器中使用顶点纹理。
构造函数会自动设置下面的所有属性。

属性（Properties）

[property:Array bones]

[page:骨骼(Bone) bones] 数组。

[property:Boolean useVertexTexture]

是否在着色器中使用一个顶点纹理，该参数在构造函数中设置。 不是所有的设备都支持浮点像素纹理。如果这个选项被设置，那么骨骼矩阵将被包装成一个纹理，并发送到着色器。 这种方法允许使用一个更大的骨骼集合。否则顶点着色将使用uniforms，这将不允许使用许多骨骼。不同设备下确切的个数也不尽相同。

[property:Array boneInverses]

一个 [page:Matrix4 Matrix4s] 数组，用来代表单个骨骼的matrixWorld逆矩阵。

[property:Integer boneTextureWidth]

顶点数据纹理的宽度。

[property:Integer boneTextureHeight]

顶点数据纹理的高度。

[property:Float32Array boneMatrices]

使用顶点纹理时，保存骨骼数据的数组缓存。

[property:DataTexture boneTexture]

使用顶点纹理时保持骨骼数据的 [page:数据纹理(DataTexture)]。

方法（Methods）

[method:null calculateInverses]()

生成 boneInverses。

[method:null pose]()

返回基本姿势的骨架。

[method:null update]()

在骨骼发生变化后更新 [page:Float32Array boneMatrices] 和 [page:数据纹理(DataTexture) boneTexture]。 该方法由 [page:WebGL渲染器(WebGLRenderer)] 自动调用，如果骨架被一个 [page:蒙皮网孔(SkinnedMesh)] 所使用的话。

[method:Skeleton clone]()

克隆该骨架对象。
