线条（[name]）

一个连续线。

例子

var material = new THREE.LineBasicMaterial({ color: 0x0000ff }); var geometry = new THREE.Geometry(); geometry.vertices.push( new THREE.Vector3( -10, 0, 0 ), new THREE.Vector3( 0, 10, 0 ), new THREE.Vector3( 10, 0, 0 ) ); var line = new THREE.Line( geometry, material ); scene.add( line );

构造器（Constructor）

[name]( [page:几何模型(Geometry) geometry], [page:材料(Material) material] )

geometry — 表示线段的顶点。
material — 线条材料。默认为 [page:基础线条材料(LineBasicMaterial) LineBasicMaterial].
如果没有提供材料，将创建一个随机材料，并分配给对象。

属性（Properties）

[property:Geometry geometry]

表示线段的顶点。

[property:Material material]

线条材料。

方法（Methods）

[method:Array raycast]( [page:Raycaster raycaster], [page:Array intersects] )

得到光线投射和该线条之间的交点。 [page:Raycaster.intersectObject] 将调用该方法。

[method:Line clone]()

返回该线条对象及其后代的克隆。
