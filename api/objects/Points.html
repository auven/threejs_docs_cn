点（[name]）

以可变大小的点形式来显示粒子的类。比如，如果使用 [page:WebGL渲染器(WebGLRenderer)]，粒子将使用 GL_POINTS 来显示。

构造器（Constructor）

[name]( [page:几何模型(Geometry) geometry], [page:材料(Material) material] )

geometry — [page:几何模型(Geometry)] 的一个实例。
material — [page:材料(Material)] 的一个实例(可选)。

属性（Properties）

[property:Geometry geometry]

一个 [page:几何模型(Geometry)] 实例，其中每个顶点指定系统中的一个粒子的位置。

[property:Material material]

一个 [page:材料(Material)] 实力， 定义对象的外观。默认的是随机色彩的 [page:点材料(PointsMaterial)]。

方法（Methods）

[method:Array raycast]( [page:Raycaster raycaster], [page:Array intersects] )

得到光线投射和该线条之间的交点。 [page:Raycaster.intersectObject] 将调用该方法。

[method:Points clone]()

返回此点对象及其后代的克隆。
